FRONTEND

Es la parte de una aplicación que interactúa con los usuarios, es conocida como el lado del cliente. Básicamente es todo lo que vemos en la pantalla cuando accedemos a un sitio web o aplicación: tipos de letra, colores, adaptación para distintas pantallas(RWD), los efectos del ratón, teclado, movimientos, desplazamientos, efectos visuales… y otros elementos que permiten navegar dentro de una página web. Este conjunto crea la experiencia del usuario.


Fundamentos

En el pasado, los sitios web eran sitios de texto simple y estático con un poco de formato y tal vez incluso algo de animación. Todo eso fue gracias a HTML y CSS.

Algunas de las tecnologías centrales utilizadas en el desarrollo web front-end incluyen:


HTML: el organizador


HTML es la forma en que se organizan todos los sitios en la web, por lo que no puede vivir sin el desarrollo de front-end.


HTML5 es la última especificación HTML.


CSS: el estilista


Hojas de estilo en cascada (CSS) es la forma en que los desarrolladores agregan estilo y efectos a un sitio web. Los estilos se pueden agregar globalmente y luego superponerse sin cambiar ese estilo fundamental que se aplica a todo un sitio.


CSS siempre está evolucionando.


Muchos desarrolladores de aplicaciones para usuario utilizan un preprocesador de CSS para hacer que escribir muchas líneas de CSS sea aún más rápido.


Los marcos de front-end CSS como Foundation o Bootstrap pueden ayudar a crear sitios web pulidos en un instante.
